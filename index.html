Стилизация

  • В разметке сначала input, потом label
  • Прячем input (.visually-hidden)
  • Стилизуем селектор input + label::before
  • Стилизуем селектор input:checked + label::before
  • Добавляем tabindex
  • Убираем фокус у спрятанного input
  • Добавляем aria-controls
  • Добавляем JS-обработчик клавиатурных нажатий

file

file



Стилизуем file


  • Скрываем type="file" (.visually-hidden)
  • Стилизуем label и устанавливаем ему role="button"
  • Устанавливаем tabindex="0" для label
  • Устанавливаем tabindex="-1" для type="file"
  • Устанавливаем aria-controls="file_id" для label
  • Стилизуем label:focus
  • JS для клавиатурных нажатий на label

За пределами боли

::placeholder

Стилизация ::placeholder



  ::-webkit-input-placeholder { color: #222; }
  ::-moz-placeholder          { color: #222; } /* Firefox 19+ */
  :-moz-placeholder           { color: #222; } /* Firefox 18- */
  :-ms-input-placeholder      { color: #222; }
  ::placeholder               { color: #222; }
